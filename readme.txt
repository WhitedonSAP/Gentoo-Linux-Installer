---------------------------------------------------------------------

1. Into the script directory select at least one mirror http/https with:

---> root # mirrorselect -i -o >> mirrors.conf

---------------------------------------------------------------------

2. Add permission to exec the installer:

---> root # chmod +x Gentoo-Linux-Installer.sh

---------------------------------------------------------------------

3. Run the installer:

---> root # ./Gentoo-Linux-Installer.sh
                         or
---> root # sh Gentoo-Linux-Installer.sh
