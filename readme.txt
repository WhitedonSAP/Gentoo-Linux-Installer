****************************** Official readme for Gentoo-Linux-Installer (GLI) ******************************
*                                                                                                            *
************************* Please connect to internet before execute the installation!!! **********************
*                                                                                                            *
******************************************* (RUN IT WITH ROOT) ***********************************************
*                                                                                                            *
*  ######################################### To the installer: ############################################  *
*  #                                                                                                      #  *
*  #  1. Select 'ONLY' one mirror first for download the files (Only https or http mirror for while!!!):  #  *
*  #                                                                                                      #  *
*  #  root # mirrorselect -i -o >> /etc/portage/make.conf                                                 #  *
*  #                                                                                                      #  *
*  ########################################################################################################  *
*  #                                                                                                      #  *
*  #  2. Add permission to exec the installer:                                                            #  *
*  #                                                                                                      #  *
*  #  root # chmod +x Gentoo-Linux-Installer.sh                                                           #  *
*  #                                                                                                      #  *
*  ########################################################################################################  *
*  #                                                                                                      #  *
*  #  3. Run the installer:                                                                               #  *
*  #                                                                                                      #  *
*  #  root # ./Gentoo-Linux-Installer.sh                                                                  #  *
*  #                     or                                                                               #  *
*  #  root # sh Gentoo-Linux-Installer.sh                                                                 #  *
*  #                                                                                                      #  *
*  ########################################################################################################  *
*                                                                                                            *
*  ############################################# To chroot: ###############################################  *
*  #                                                                                                      #  *
*  #  1. Add permission to exec the files:                                                                #  *
*  #                                                                                                      #  *
*  #  root # chmod +x chroot/Chroot.sh chroot/Umount.sh                                                   #  *
*  #                                                                                                      #  *
*  ########################################################################################################  *
*  #                                                                                                      #  *
*  #  2. Run the script to chroot:                                                                        #  *
*  #                                                                                                      #  *
*  #  root # ./chroot/Chroot.sh                                                                           #  *
*  #                or                                                                                    #  *
*  #  root # sh chroot/Chroot.sh                                                                          #  *
*  #                                                                                                      #  *
*  ########################################################################################################  *
*  #                                                                                                      #  *
*  #  3. To umount:                                                                                       #  *
*  #                                                                                                      #  *
*  #  root(chroot) # exit                                                                                 #  *
*  #                                                                                                      #  *
*  #  root # ./chroot/Umount.sh                                                                           #  *
*  #                or                                                                                    #  *
*  #  root # sh chroot/Umount.sh                                                                          #  *
*  #                                                                                                      #  *
